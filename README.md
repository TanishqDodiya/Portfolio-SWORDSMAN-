# Tanishq Dodiya Portfolio

A stylized, anime-inspired portfolio landing page built as a single static HTML experience.

## Overview

This project is a personal portfolio page for **Tanishq Dodiya** presented as a narrative "swordsman" journey. It uses modern layout and animation techniques to showcase education, skills, projects, achievements, and contact details in a visually immersive format.

## Files

- `tanishq_zoro_portfolio.html` — main portfolio page
- `assets/` — supporting assets used by the portfolio page

## Features

- Animated hero introduction with custom cursor and visual effects
- Responsive navigation with desktop and mobile menu support
- Education section styled as training chapters
- Skills section presented as sword styles with skill tags
- Project / mission showcase with featured and bounty-style case studies
- Achievements and timeline sections for personal accomplishments
- Contact section with stylized contact form and social links
- Thematic layout inspired by a storytelling and anime warrior journey

## Technologies

- HTML
- CSS
- JavaScript
- Google Fonts
- External animation libraries:
  - GSAP
  - ScrollTrigger
  - Lenis

## Usage

1. Place `tanishq_zoro_portfolio.html` and the `assets/` folder in the same directory.
2. Open `tanishq_zoro_portfolio.html` in a web browser.
3. No build process is required; this is a static portfolio page.

## Notes

- The contact form is included for presentation and may require backend handling if you want actual message delivery.
- Update the portfolio content, asset links, or project details as needed to customize the page.

## Customization

To personalize the portfolio, modify the following directly in `tanishq_zoro_portfolio.html`:

- hero name, quote, and roles
- education entries
- skill categories and tags
- project mission details
- achievements and timeline items
- contact links and social references
